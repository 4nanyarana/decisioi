# Decisio!

A simple decision-making app because I literally can never choose what to eat with my friends. 
We always spend 30 minutes arguing over picking a place, so I built a wheel to just decide for us.

## How it works
1. Put in your options
2. Spin the wheel
3. Let it decide. No arguing allowed.

## Tech stack
- React (frontend)
- Express + Node.js (backend)
- MongoDB (to save the results... just in case someone claims the wheel is rigged)

## How to run locally
Run `npm run build` in the `client` folder, and then run `node server.js` in the main folder. Then open `http://localhost:3000`.
